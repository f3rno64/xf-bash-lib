# f3rno64/xf-bash-lib

A utility bash library for my personal dotfiles, sourcing utilities like `nvm`
and `rvm` based on their availability, and configuring the bash environment for
the host OS.

## Supports

- Linux
- MacOS
- Termux (Android)

##

Adds many aliases, such as `ga` for `git add -p`.
